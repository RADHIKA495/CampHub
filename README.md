#  CampHub

CampHub is a full-stack campground review platform that allows users to create, view, review, and manage campgrounds.  
It implements secure authentication, authorization, and RESTful architecture.


##  Features

- User authentication (Register / Login / Logout)
- Authorization (only owners can edit or delete campgrounds and reviews)
- Campground CRUD operations
- Reviews and ratings
- Flash messages for success and error handling
- Form validation using Joi
- RESTful routing
